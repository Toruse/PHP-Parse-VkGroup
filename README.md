#PHP-Parse-VkGroup

    class\ - содержит класс для работы парсера
    config\ - содержит конфигурационные файлы
    file\ - содержит файлы с изображениями
    phpQuery\ - содержит файлы phpQuery
    hunterphotos.bat - исполняемый файл для командной строки
    hunterphotos.php - содержит скрип для парсинга ресурса
    revolife.sql - dump базы данных.

###Принцип работы.

При первом запуск парсер сканирует все страницы ресурса, и добавляет в базу  данных, после чего загружает изображения на сервер. 
При вторичном и последующих запусках парсер, он находит последние добавленные посты и добавлять их в базу данных.
