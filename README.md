--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
Написание и тестирование скрипта проводилось на Denwerе под Windows 7.
В консольном режиме скрипт отработал хорошо.
--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
Принцип работы.
При первом запуск парсер сканирует все страницы ресурса, и добавляет в базу 
данных, после чего загружает изображения на сервер.
При вторичном и последующих запусках парсер, он будет находит последние 
добавленные посты и добавлять их в базу данных.
--------------------------------------------------------------------------------
